# Stock Price Observer Mini Project

## Overview

This mini project shows Observer using stock price alerts.

Investors subscribe to a stock.

When price changes, all subscribed investors receive an update.

## Pattern Mapping

Subject:

- Stock

Observer:

- Investor

Concrete Observer:

- MobileInvestor

## How to Run

```bash
javac src/DesignPatterns/behavioural/observer/miniProject/App.java
java -cp src DesignPatterns.behavioural.observer.miniProject.App
```
