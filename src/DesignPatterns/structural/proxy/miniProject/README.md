# Secure Document Proxy Mini Project

## Overview

This mini project shows Proxy using secure document access.

The real document contains sensitive data.

The proxy checks user role before opening it.

## Pattern Mapping

Subject:

- Document

Real Subject:

- ConfidentialDocument

Proxy:

- SecureDocumentProxy

Client:

- App

## How to Run

```bash
javac src/DesignPatterns/structural/proxy/miniProject/App.java
java -cp src DesignPatterns.structural.proxy.miniProject.App
```
